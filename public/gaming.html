üìö Webhook Setup Guide

üéÆ Steam API

Method: REST API (Polling)

  1. Get API key from steamcommunity.com/dev
  2. Set STEAM_API_KEY environment variable
  3. Poll API endpoints at regular intervals
  4. Available data: profile, games, achievements

üéØ Xbox Live (OpenXBL)

Method: Third-party API

  1. Sign up at xbl.io
  2. Get API key from dashboard
  3. Set XBOX_API_KEY environment variable
  4. Available data: profile, achievements, friends

üéÆ PlayStation Network

Method: Unofficial APIs

  1. No official public API available
  2. Use PSNProfiles.com API (unofficial)
  3. Or use TrueTrophies API
  4. Alternative: Web scraping

üí¨ Discord Webhooks

Method: Incoming Webhooks

  1. Go to Server Settings ‚Üí Integrations
  2. Create new webhook
  3. Copy webhook URL
  4. Set DISCORD_WEBHOOK_URL environment variable

üì∫ Twitch EventSub

Method: Webhooks + API

  1. Register app at dev.twitch.tv
  2. Get Client ID and Client Secret
  3. Set TWITCH_CLIENT_ID and TWITCH_CLIENT_SECRET
  4. Subscribe to EventSub topics

‚ö° NVIDIA GeForce

Method: Not Available

  1. No public API available
  2. NVIDIA does not provide GeForce Now API
  3. Consider web scraping (against TOS)
  4. Monitor NVIDIA developer site for updates
